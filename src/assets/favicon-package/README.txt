# Favicon Package

Generated with NeoWeb Tools Favicon Generator

## Included Files
- favicon-16x16.png: Browser tab (standard)
- favicon-32x32.png: Browser tab (Retina)
- favicon-48x48.png: Windows site icon
- favicon-64x64.png: Windows desktop
- favicon-128x128.png: Chrome Web Store
- favicon-180x180.png: Apple Touch Icon
- favicon-192x192.png: Android Chrome
- favicon-512x512.png: PWA splash screens
- favicon-html-snippet.html: HTML code to add to your website

## Usage
1. Upload all PNG files to your website's root directory
2. Add the HTML snippet to your <head> section

## Browser Support
- favicon-16x16.png & favicon-32x32.png: All browsers
- apple-touch-icon.png (180x180): iOS Safari
- favicon-192x192.png & favicon-512x512.png: Android Chrome, PWA
